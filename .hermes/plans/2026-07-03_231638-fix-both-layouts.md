# Fix Both Layout Issues — Timer Card + Modal

> **For Hermes:** Execute directly — 3 tasks, 2 files.

**Goal:** Fix two layout issues: (1) edit button overlapping note text on timer card, (2) modal form inputs not aligned with padding edges.

**Tech Stack:** Svelte 3, CSS

---

### Task 1: Fix edit button overlap on timer card

**Files:** `src/App.svelte`

**Fix:** Add `padding-right: 48px; min-height: 40px` to the content section so the absolute-positioned edit button has its own space.

Change line ~748:
```svelte
<div class="section" style="position: relative;">
```
To:
```svelte
<div class="section" style="position: relative; padding-right: 48px; min-height: 40px;">
```

**Commit:** `fix: add padding-right to card section to prevent edit button overlap`

---

### Task 2: Fix modal body row negative margins

**Files:** `src/components/EditTimerModal.svelte`

**Problem:** Bootstrap `.row` has negative `margin-left/right: -15px`, which makes the inputs extend past the `.modal-body` padding.

**Fix:** Remove the `row` class and use a custom flex row instead.

Replace (lines 135-160):
```svelte
        <div class="row" style="margin-bottom: 10px;">
          <div class="col-sm-6">
```
With:
```svelte
        <div style="display: flex; gap: 10px; margin-bottom: 10px;">
          <div style="flex: 1;">
```

And close by replacing (after the items div closes):
```svelte
          </div>
        </div>
```

**Commit:** `fix: remove bootstrap row negative margins from modal body`

---

### Task 3: Fix modal header alignment with multi-line title

**Files:** `src/components/EditTimerModal.svelte`

**Problem:** Title "Edit Timer N" + countdown wraps to 2 lines, X button centers on 2-line height instead of aligning with top line.

**Fix:** Change header alignment from `align-items: center` to `align-items: flex-start`, add `line-height: 1.3` for tighter text.

CSS change:
```css
  .modal-header {
    display: flex;
    align-items: flex-start;
    justify-content: space-between;
    padding: 16px 20px 0 20px;
  }
```

**Commit:** `fix: align modal header X button to top when title wraps`
