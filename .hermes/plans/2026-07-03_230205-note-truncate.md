# Truncate Long Notes on Timer Card

> **For Hermes:** Execute directly — 1 task, 1 file.

**Goal:** Prevent long notes from breaking the card layout. Notes ≤ 40 chars display fully; longer notes truncate with "...". Full text always shown in edit modal.

**Architecture:** Add inline conditional truncation in the Svelte template where note text is rendered. Zero logic change needed elsewhere.

**Tech Stack:** Svelte 3, CSS `word-break` as defensive fallback

---

### Task 1: Truncate note display on card

**Objective:** Truncate notes > 40 characters with ellipsis on the timer card

**Files:**
- Modify: `src/App.svelte` — note display line in template

**Step 1: Add truncation in template**

Current (line ~756):
```svelte
<span class="justinfo" style="color: #888; font-style: italic;">{timer.note}</span>
```

Replace with:
```svelte
<span class="justinfo" style="color: #888; font-style: italic;">
  {timer.note.length > 40 ? timer.note.slice(0, 40) + '...' : timer.note}
</span>
```

**Step 2: Add defensive word-break CSS**

In the `<style>` section, add:
```css
  .justinfo {
    word-break: break-word;
    overflow-wrap: break-word;
  }
```

This prevents any edge case where mid-word overflow could still break the layout.


Expected: success
