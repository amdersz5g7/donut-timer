# Sync Markdown Files — Version & Date

> **For Hermes:** Batch update via sed — 1 command, 7 files.

**Goal:** All project .md files have consistent version (`v26.07`) and last-updated date (`2026-07-04`) matching README.md.

**Files (7):**
- `AUTOSCROLL_FIX.md`
- `EDIT_BUTTON_COLOR.md`
- `FLASH_ANIMATION.md`
- `IMPROVEMENTS.md`
- `TESTING_CHECKLIST.md`
- `TIMEINFO_ENHANCEMENT.md`
- `src/utils/README.md`

**Command:**

```bash
cd /home/i3wm/Documents/github/donut-timer-claude && \
for f in AUTOSCROLL_FIX.md EDIT_BUTTON_COLOR.md FLASH_ANIMATION.md IMPROVEMENTS.md TESTING_CHECKLIST.md TIMEINFO_ENHANCEMENT.md src/utils/README.md; do
  sed -i \
    -e 's/\*\*Last Updated:\*\* 2025-12-05/\*\*Last Updated:\*\* 2026-07-04/' \
    -e 's/\*\*Version:\*\* v25\.12.*/\*\*Version:\*\* v26.07/' \
    "$f"
done && git add -A && git commit -m "docs: sync version to v26.07 across all markdown files"
```
