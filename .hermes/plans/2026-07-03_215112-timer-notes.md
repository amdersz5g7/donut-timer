# Add Notes Feature to Donut Timer

> **For Hermes:** Use subagent-driven-development skill to implement this plan task-by-task.

**Goal:** Allow users to attach a text note to each timer — visible on the card, editable in edit mode, persisted to localStorage.

**Architecture:** Add `note` field to timer objects. Pipe it through `TimerForm` → `addTimer()`, display it on the timer card, and include it in edit form. Backward compatible — existing timers without `note` default to empty string.

**Tech Stack:** Svelte 3, localStorage (via `persistent.js` writable store)

---

## Current Data Model

```js
{
  tid: number,
  remove: false,
  text: "Timer N",
  start_at: "HH:MM:SS",
  finish_at: "HH:MM:SS",
  maxminute: number,
  items: number,
  start_full: Date,
  finish_full: Date,
  done: false,
  note: ""  // ← NEW
}
```
